# Hospital Patient Dashboard

This project presents an interactive dashboard built using Power BI to analyze hospital patient data. The dashboard provides a clear overview of patient information, medical records, hospital charges, and medicine usage.

## Project Overview
The Hospital Patient Dashboard is designed to help hospital staff easily monitor patient information and hospital operations. It provides a centralized view of important medical and administrative data through interactive visualizations.

The dashboard allows users to view patient details, track medicine consumption, and analyze hospital charges in order to support better decision-making in healthcare management.

## Project Objectives
The main objectives of this project include:

- Monitoring patient information and medical records  
- Tracking medicine usage and sales  
- Analyzing hospital charges by different service types  
- Providing a clear overview of hospital performance indicators  
- Supporting data-driven decision making in healthcare

## Dashboard Features

### Patient Information Panel
The dashboard displays detailed patient information including:
- Patient name
- Age
- Address
- Diagnosis
- Assigned doctor
- Department
- Bed ID and bed status

### Hospital Metrics
Key hospital indicators are displayed such as:
- Total bill amount
- Admission date
- Discharge status
- Total medicine sales quantity

### Medicine Tracking
A table visualizes medicine tracking by day and month to monitor medication usage trends.

### Medicine Sales Analysis
A bar chart shows medicine sales quantity for different medicines such as:
- Amoxicillin
- Amlodipine
- Metformin
- Ranitidine

### Hospital Charges Analysis
A chart analyzes hospital spending based on charge types including:
- Surgery charges
- Room charges
- Test charges
- Doctor fees
- Medicine charges

## Tools Used
